## git log

```bash
git log 
```

^112cbc

Diesem Befehl wird verwendet, um **alle** Save Points (commits) anzuzeigen.

>[!tip] 
>`git log --oneline` mach das selbe wie `git log`,  aber in eine kleinere Version.
