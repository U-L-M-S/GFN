## übersich Netzwerk 
### Übersicht
ich fühle den Befehl `ip a` an meine Rechner, Laptop und Handy. Dann erhalte ich folgende Informationen:

PC:
```bash
IPv4:
	192.168.1.204
Subnet Mask:
	255.255.255.0
Gateway:
	192.168.1.1
```

Laptop:
```bash
IPv4:
	192.168.1.25
Subnet Mask:
	255.255.255.0
Gateway:
	192.168.1.1 
```


Handy:
```bash
IPv4:
	192.168.1.69
Subnet Mask:
	255.255.255.0
Gateway:
	192.168.1.1
```
